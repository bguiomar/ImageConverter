#

This project implement a web application that allow you to upload an image and visualize it in greyscale. Of course you can also download it.

## Quick start

### Start backend

1. Got to backend folder: `cd backend`
2. Install dependecies: `npm install`
3. Run server: `node index.js`

### Start frontend

1. Got to frontend folder: `cd frontend`
2. Install dependecies: `npm install`
3. Run client: `npm start`
4. Now you can visit [http://localhost:3000](http://localhost:3000)
